==========
配置
==========

DeerU的所有配置都采用json格式文件进行配置，你可以在 http://127.0.0.1:8000/admin/app/config/ 中查看修改配置


.. note::

   需要注意的是配置文件中bool值采用0，1表示，0-flase, 1-true
