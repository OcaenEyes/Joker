=============
通用配置
=============

配置一些其他的东西，主题、插件的一些专业配置也会放到里面


theme
--------

    * 是否必须 : 是
    * 值类型 : 字符串
    * 默认值 : base_theme

    使用的主题


baidu_auto_push
-----------------

    * 是否必须 : 是
    * 值类型 : bool
    * 默认值 : 0


    base_theme的配置，是否自动推送url到百度  
    （使用的是百度的自动推送脚本，不过这个脚本貌似已经好久没维护，不能正常运行了）
